This is the solution for "Problem 1: Airport Baggage".

How to run?
- Please import "barclays-project" in your eclipse as it's a maven project. Main class is "AirportApp" which you can run directly in eclipse.
- There is a "input.txt" file which is a input file. You can modify that input file as per your requirements. And you need to pass this input file location as the command line arguments.
